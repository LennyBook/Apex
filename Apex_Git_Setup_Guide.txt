[Instructions]
- Use this guide to setup local Git in ~/Apex, ignore backups/archives/3rd_Party/Installers, track Session_Files and project hubs/tasks, sync to GitHub for Grok access.
- Use unversioned filenames (e.g., Apex_Operating_Files_List.txt) with internal Version/Last_Known_Stable.
- Commit/push only key versions (latest and stable) to GitHub; intermediates stay local.
- For In_Progress WIP: Add to Git when ready, commit/tag/push (latest/main, stable tag), update AOFL [Git_Tracked_Files] with raw links.
- Use British English, FMT008 formatting.

[Git_Setup_Steps]
Description: Step-by-Step Terminal Commands for Local Git and GitHub Mirror
1. Navigate to Apex Root: cd ~/Apex
2. Initialize Git: git init
3. Create/Update .gitignore: nano .gitignore (paste the updated .gitignore content from Grok's previous response, save).
4. Stage .gitignore: git add .gitignore
5. Stage Session_Files: git add Apex/System/Session_Files
6. Stage Project Hubs/Tasks: git add Apex/Projects/Writing/Writing_Hub.txt
git add Apex/Projects/Writing/Writing_Tasks_Snapshot.txt
git add Apex/Projects/Video_Games/Video_Game_Hub.txt
git add Apex/Projects/Video_Games/Video_Games_Tasks_Snapshot.txt
git add Apex/Projects/Video_Games/Deep_Sea_NN_Wars/Deep_Sea_NN_Wars_Hub.txt
git add Apex/Projects/Video_Games/Deep_Sea_NN_Wars/Deep_Sea_NN_Wars_Tasks_Snapshot.txt
git add Apex/Projects/Video_Games/Futures_Present_Past/Futures_Present_Past_Hub.txt
git add Apex/Projects/Video_Games/Futures_Present_Past/Futures_Present_Past_Tasks_Snapshot.txt
git add Apex/Projects/Home_Household/Home_Household_Hub.txt
git add Apex/Projects/Home_Household/Home_Household_Tasks_Snapshot.txt
git add Apex/Projects/Websites/Websites_Hub.txt
git add Apex/Projects/Websites/Websites_Tasks_Snapshot.txt
git add Apex/System/Session_Files/Len_Master_Hub.txt
7. Commit Initial: git commit -m "Initial commit - Session_Files and project hubs/tasks, unversioned with internal versions"
8. Tag Stable: git tag stable-initial
9. Set Remote (Private Repo): git remote add origin https://github.com/LennyBook/Apex.git
git branch -M main
10. Push: git push -u origin main --tags
11. Future Update (e.g., new version of Apex_Operating_Files_List.txt):
Edit file, git add Apex/System/Session_Files/Apex_Operating_Files_List.txt
git commit -m "Update AOFL internal v1.1 with new hub links"
git tag latest-v1.1
git push origin main --tags
12. Rollback (e.g., to stable-initial): git checkout stable-initial -- Apex/System/Session_Files/Apex_Operating_Files_List.txt
git add Apex/System/Session_Files/Apex_Operating_Files_List.txt
git commit -m "Reverted AOFL to stable v1.0 due to issues"
git tag restored-v1.0
git push origin main --tags

[In_Progress_Workflow]
